CRÉDITOS Y ATRIBUCIONES
========================================

PROYECTO ORIGINAL
========================================

QSO Map
Autor: Ian Renton
Website: https://ianrenton.com
Repositorio: https://git.ianrenton.com/ian/qsomap.git
Licencia: MIT License

Descripción:
Sistema de visualización de QSOs en mapa interactivo usando Leaflet,
con soporte para múltiples fuentes de datos (LOTW, Clublog) y
lookups de grid automáticos.

========================================
MEJORAS Y ADAPTACIONES
========================================

Implementadas para LU2MET por Tomás Duca
Repositorio: https://github.com/tomduca/qsomap

Principales mejoras:
- Sincronización automática con Clublog (primaria) y LOTW (fallback)
- Cache incremental para mejor performance
- Preservación de grids manuales de RUMlogNG/Clublog
- Scripts portables para hosting compartido sin SSH
- Margen de seguridad de 2 días en sync incremental
- Fix de colores de marcadores por banda
- Documentación completa en español para deployment en cPanel

========================================
BIBLIOTECAS Y SERVICIOS UTILIZADOS
========================================

FRONTEND:
- Leaflet.js - Biblioteca de mapas interactivos
- jQuery - Manipulación DOM
- Esri.NatGeoWorldMap - Mapa base

BACKEND:
- PHP 7.4+ - Procesamiento server-side
- cURL - Comunicación con APIs

APIs EXTERNAS:
- Clublog API - Fuente primaria de QSOs
- LOTW (Logbook of The World) - Fuente fallback de QSOs
- HamQTH - Lookup de grids (primario)
- Spothole - Lookup de grids (fallback)

UTILIDADES:
- ham.js - Funciones de radio amateur (Ian Renton)
- ui-ham.js - UI helpers para ham radio (Ian Renton)
